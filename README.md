# OverFit Academia

Projeto front-end simples para a página inicial da academia OverFit.

## Visão Geral
Este repositório contém um protótipo estático com a página principal e ativos de imagem. Ideal para demonstrar o layout inicial e identidade visual.

## Estrutura do Projeto
- `index.html` — página principal
- `README.md` — documentação do projeto
- `hero-bg.png`, `logo-overfit.png` — imagens/ativos

## Começando (Quick Start)
1. Abra o projeto no VS Code.
2. Para visualizar localmente, abra `index.html` no navegador ou use a extensão Live Server.

## Desenvolvimento
- Edite `index.html` para alterar o conteúdo/estrutura da página.
- Coloque imagens na mesma pasta para que os caminhos relativos funcionem.

## Fluxo Git recomendado
- Crie uma branch para novas funcionalidades: `git checkout -b feat/nome-da-feature`
- Faça commits claros: `git commit -m "feat: adiciona seção sobre horários"`
- Suba para o remoto e abra Pull Request no GitHub.
